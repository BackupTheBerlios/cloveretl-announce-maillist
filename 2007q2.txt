Dear CloverETL user !

We have just released CloverETL version 2.1.2 which fixes several 
bugs/issues discovered
recently.

These are the most important changes this new version brings:

Changes/bug fixes:
+ DBOutputTable component now properly recognizes empty cloverField parameter
+ DataGenerator component publics its parameter setters
+ support for sequences and lookup tables was added into clover transformation language
+ simple sequence now returns valid current value

You may download the new release through following 
link: http://developer.berlios.de/project/showfiles.php?group_id=618
